# Test Artifacts — Sumiran (JAAP-LEDGER)

This folder holds all testing artifacts generated by Claude Code STLC Agents.

## Structure

```
test-artifacts/
  scenarios/      — Test Scenarios (what to test)
  test-cases/     — Detailed Test Cases (how to test)
  test-data/      — Test Data sets (data to use)
  defects/        — Defect Reports (what broke)
  reports/        — Test Execution Reports (summary)
```

## How Artifacts Are Created

Each artifact is generated by a specific agent prompt described in `docs/learning/`.

| Artifact Type | Generated By | Learning Module |
|--------------|-------------|-----------------|
| Scenarios | Test Scenario Agent | Module 03 |
| Test Cases | Test Case Agent | Module 04 |
| Test Data | Test Data Agent | Module 05 |
| Defects | Defect Logging Agent | Module 08 |
| Reports | Test Report Agent | Module 09 |

## Naming Conventions

- Scenarios: `[module]-scenarios.md` (e.g., `today-card-scenarios.md`)
- Test Cases: `[module]-test-cases.md`
- Defects: `DEF-SUMIRAN-[NNN].md` (e.g., `DEF-SUMIRAN-001.md`)
- Reports: `test-report-[YYYY-MM-DD].md`
